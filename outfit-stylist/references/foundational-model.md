# Foundational Outfit Model

Use this as the reasoning spine. Do not expose the theory names unless the user asks for deeper reasoning.

## Four Layers

```text
Philosophical core: truth / ease / readability / agency
Root functions: protection / boundary / role / social order / expression
Aesthetic judgment: propriety / harmony / degree / form / qi / context / flavor
Practical inputs: person / weather / occasion / clothes / intent
```

## Final Check

- Truth: Does this feel like a plausible version of the user?
- Ease: Can the user move, sit, commute, talk, eat, and focus without fussing over the clothes?
- Readability: Will the scene understand the intended signal?
- Agency: Does the outfit help the user do the day?

## Root Functions

- Protection: temperature, weather, movement, friction, exposure.
- Boundary: how much the body is shown, structured, softened, distanced, or made approachable.
- Role: worker, host, guest, date, interviewer, traveler, friend, performer.
- Social order: profession, city, hierarchy, relationship, age group, cultural norms.
- Expression: one personal note without making the outfit unreadable.

## Aesthetic and Cultural Judgment

- Propriety / 礼: respect the occasion, relationship, role, and dress code.
- Harmony / 和: align body, garment, environment, color, texture, and mood.
- Degree / 度: calibrate formality, trendiness, exposure, decoration, and effort.
- Form / 形: check proportion, line, silhouette, visual weight, and rhythm.
- Qi / 气: name the state created: relaxed, sharp, gentle, bright, calm, dignified, intimate, authoritative.
- Context / 境: fit weather, city, venue, time, movement, and culture.
- Flavor / 味: add a controlled memorable note.

## Conflict Priority

```text
safety and body comfort
> occasion propriety
> user truth and confidence
> aesthetic coherence
> trend relevance
```

Trend is a reference layer, not the decision layer.
