# Trend Reference Guidelines

Current trends are optional references, not decision makers.

## Use Trend Context When

- The user asks "现在流行什么", "想要不显过时", or "帮我加一点当下感".
- The outfit risks looking dated.
- A small trend update can refresh existing clothes.

## Do Not

- Let trend override comfort, propriety, truth, or execution.
- Recommend trend-driven purchases without repeated use value.
- Present social media aesthetics as universal.

## Trend Translation

Translate trends into small usable moves:

- color update
- shoe shape
- trouser width
- layering method
- bag/accessory scale
- material mix
- hair/makeup finish

Say which part of the trend to borrow and which part to skip.
